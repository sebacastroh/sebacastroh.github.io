# Programación científica en Python

La objetivo de este proyecto es transmitir mis conocimientos de programación científica. La idea es cubrir la mayor cantidad de conceptos, desde las cosas más simples como gráfico de funciones hasta tareas más complicadas (pero no imposibles) como la optimización de códigos. La idea es transmitir este conocimiento a través de ejemplos prácticos, además de aprender nuevos algoritmos.
